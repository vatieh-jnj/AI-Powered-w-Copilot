# 📦 Supply Chain Issue Analysis

## 🚀 Overview
This project analyzes critical supply chain disruptions impacting production continuity and FDA compliance.  
It combines data analysis, stakeholder alignment, and actionable remediation planning using Microsoft Copilot.

---

## 📁 Project Structure

```

.
├── docs/            # Written analysis and summaries
├── data/            # Source dataset
├── presentation/    # Executive slide deck
├── meeting/         # Meeting artifacts (agenda, invite)
├── prompts/         # Copilot prompts used to generate outputs
└── README.md

```
